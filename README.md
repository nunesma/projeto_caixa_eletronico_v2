## Primeiro desafio
- Crie um novo projeto no IntelliJ
- Crie uma classe com o método main
- Escreva um algoritmo que simula a funcionalidade de saque:
- A pessoa informa o nome e o valor a ser sacado
- O sistema exibe se o saque pode ou não ser realizado, de acordo com as regras de negócio


## Objetivos
- Criação de projeto no IntelliJ
- Declaração de variáveis no Java
- Utilização de tipos primitivos e String no Java
- Leitura de dados do teclado
- Controle de fluxos com if/else
- Cálculos matemáticos e operadores lógicos

## Regras de Negócio
- Valor máximo que pode ser sacado é R$1.000,00
- Valor mínimo que pode ser sacado é R$10,00
- Caixa eletrônico somente possui cédulas de R$10, R$20, R$50 e R$100
- Valor do saque não deve ultrapassar o saldo da conta
- Caso o saque possa ser realizado, exibir a mensagem:
    - Olá NOME! O saque de VALOR foi realizado com sucesso!
- Caso o saque não possa ser realizado, exibir a mensagem:
    - Olá NOME! O saque de VALOR não pode ser realizado: MOTIVO
